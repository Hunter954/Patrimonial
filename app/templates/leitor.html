Leitor (Código de Barras / QR)

No PC use leitor USB. No celular, clique em Usar câmera.
Cadastrar manualmente
Escanear etiqueta
Para abrir a câmera, o site precisa estar em HTTPS (ou localhost). Verifique se a URL começa com https://.
Leitor USB: clique no campo e escaneie (muitos leitores enviam Enter no final).
Celular: clique em Usar câmera e aponte para o QR/código de barras.
Permita o acesso à câmera quando o navegador pedir.
Como funciona
  1. Você escaneia a etiqueta (USB ou câmera)
  2. O sistema procura o patrimônio pelo código
  3. Se existir: abre direto para editar
  4. Se não existir: abre o cadastro com o código preenchido
Dicas rápidas
  • Evite abrir no navegador “dentro do Instagram/WhatsApp”. Use Chrome/Safari.
  • No iPhone: Ajustes → Safari → Câmera (permitir) ou nas permissões do site.
  • Se aparecer duas câmeras, o sistema tenta escolher a traseira automaticamente.
